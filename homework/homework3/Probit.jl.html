Probit 🔮 Monte Carlo: Maximum Likelihood

In this homework you will implement an estimator for the well known Probit model. To remind you, a simple version of this model is defined as follows:

yi∈{0,1}Pr{yi=1|xi}=Φ(xiβ)L(β)=Πi=1NΦ(xiβ)yi(1−Φ(xiβ))1−yiβ∈Rkxi∼N([0,0,0],[100010001])k=3

where Φ denotes the standard normal cdf and where L denotes the likelihood function. Think of xi as a row vector (1,k) such that if you stack all x for all n people you get an (n,k) matrix where a row is an observation. Notice that a common way to set this up is as a latent variable model like

yi∗=xiβ+ϵiyi=1[yi∗≥0]ϵi∼N(0,1)

Question 2: Write up the log likelihood function

  • 👉 This is L=log⁡(L):

L(β)=∑yi=1log⁡Φ(xiβ)+∑yi=0log⁡(1−Φ(xiβ))

  • For numerical stability it is better to focus on the average log likelihood function, hence we divide through by the sample size n. Additionally we multiply by negative one, since Optim.jl wants to minimize functions. Let's call this the log likelihood function in what follows:

l(β)=−1n(∑yi=1log⁡Φ(xiβ)+∑yi=0log⁡(1−Φ(xiβ)))

  • write the function loglik(betas::Vector,d::Dict) which will compute the log likelihood at a given value for the β.

  • The maximum likelihood estimator will maximize that expression.

Question 5: standard errors!

We can use the either the inverse hessian or the information matrix to obtain standard errors for the maximimum likelihood estimator. Here is how we calculate the covariance matrix for the estimator in either case

The Inverse observed information matrix

cov1(βML)=1n(∂2l∂β2|β=βML)−1

We divide by n because we computed the average log likelihood all along. To get the standard errors for each β, just grab the diagonal elements of this k,k matrix with diag, and then take the square root of each element.

The inverse of the expected information matrix

The information matrix is defined as the negative of the expected value of the hessian. It comes down to this expression:

I(β)=∑i=1nϕ(si)2Φ(si)(1−Φ(si))xi′xi

Then do the same as above, i.e the standard errors are 

sqrt.(diag(inv_Info(betas,d)))

at the optimal betas.

Hint

function info_mat(betas::Vector,d)
    xbeta     = d[:X]*betas # (n,1)
    G_xbeta   = cdf.(d[:dist],xbeta)    # (n,1)
    one_G_xbeta= 1.0 .- G_xbeta
    g_xbeta   = pdf.(d[:dist],xbeta)    # (n,1)

    out = zeros(length(betas),length(betas))

    for i in 1:d[:n]
        XX = d[:X][i,:] * d[:X][i,:]'   #k,k
        out[:,:] = out + XX * g_xbeta[i]^2 / (G_xbeta[i] * one_G_xbeta[i])
    end
    return out
end
